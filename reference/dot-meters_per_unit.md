# Function returns the number of meters in unit (one of miles, kilometers, or meters)

Function returns the number of meters in unit (one of miles, kilometers,
or meters)

## Usage

``` r
.meters_per_unit(unit)
```

## Arguments

- unit:

  Character scalar specifying distance unit; one of `"miles"`,
  `"kilometers"`, or `"meters"`.

## Value

Numeric scalar conversion factor from the selected unit to meters.
