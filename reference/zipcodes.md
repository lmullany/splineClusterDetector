# Zipcode Location Dataset

A data set that provides latitude and longitude for each zipcode in the
United Sates

## Usage

``` r
zipcodes
```

## Format

A data frame with 42,482 rows and 11 columns:

- id:

  serial integer id (1, 2, 3, .. etc)

- zip_code:

  5 digit string for zipcode

- state:

  state abbreviation

- county:

  county name

- region:

  region name

- region_id:

  id for region

- region_name:

  region name

- pop, modified:

  undocumented

- latitude, longitude:

  numeric coordinates for zipcode
